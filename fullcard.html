File sharing

WeTransfer è un servizio di file sharing che permette di inviare file di grandi dimensioni, in formato foto, audio o video, ad altre persone tramite link. La versione gratuita ha un limite di massimo 2GB per ciascun trasferiento, ma l’abbonamento Pro consente di arrivare fino a 200GB.
